پروکسی EdgeOne

فعال — Active

این سرویس یک پروکسی معکوس (Reverse Proxy) مبتنی بر EdgeOne Pages است که درخواست‌ها را به سرور مقصد ارسال و پاسخ را به کاربر بازمی‌گرداند.

A lightweight reverse proxy running on EdgeOne Pages Edge Functions. It forwards requests to the target server and returns the response to the client.

نحوه استفاده — Usage

آدرس مقصد را در هدر x-host ارسال کنید. پروکسی مسیر و کوئری را حفظ کرده و درخواست را به مقصد ارسال می‌کند.

Send the target host in the x-host header. The proxy preserves the path and query string and forwards the request to the destination.

Example:
curl -H "x-host: example.com" https://your-domain/path

بررسی سلامت — Health Check

درخواست بدون هدر x-host یک پاسخ ساده برای بررسی سلامت سرویس برمی‌گرداند.

A request without the x-host header returns a simple health check response indicating the proxy is active.
